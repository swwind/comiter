# comiter

A comic downloader.

## Install

Simply install via `npm` or `yarn`.

```bash
# use npm
npm install comiter -g
# use yarn
yarn global add comiter
```

## Usage

```bash
comiter [options] [url]
# e.g.
comiter "https://manhua.dmzj.com/biedangounijiangle"
# Automaticly download to a new sub directory
```
