# comiter

A comic downloader.

## Install

```bash
npm install comiter -g
# or
yarn global add comiter
```

## Usage

```bash
comiter [options] [url]
# e.g.
comiter "https://manhua.dmzj.com/biedangounijiangle"
# Automaticly download to current directory
```
